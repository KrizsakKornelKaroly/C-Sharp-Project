Ebben a feladatban egy jegyfoglaló programot készítettünk, amiben az adott adatbázisból 3 táblával dolgozik, hogy pontos adatokat mutasson.
A programban le tudod kérni az összes eseményt, az adott eseményeknek elérhető jegyeit, és még le is tudsz foglalni jegyeket.
Ami a tervben volt megcsináltuk, párat consumer-friendly-vé alakítottuk, hogy kevesebb probléma legyen a program használatakor.
